# 台灣 3+3 郵遞區號查詢

即時、快速、無需驗證碼之郵遞區號線上查詢工具。
## 前端
```bash
npm i
npm run dev
```
## 資料庫
有關轉換工具與資料庫取得，請檢視 [utils/database-conventer](https://github.com/gnehs/TaiwanZipcode/tree/main/utils/database-conventer) 說明

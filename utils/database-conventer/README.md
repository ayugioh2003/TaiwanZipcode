# 資料庫轉換器
將郵局資料庫轉換為 json 格式
## 取得
- 你可以在郵局網站取得 [3+3 郵遞區號應用系統](https://www.post.gov.tw/post/internet/Download/all_list.jsp?ID=2201#dl_link_2735)
- 安裝後可以在 `C:\Zip33U\DBF\rall1.dbf` 取得資料庫檔案
- 將資料庫檔案放在 `./src` 之下
- 終端機輸入 `node index.js` 進行轉換
## 資料庫授權條款
